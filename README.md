# Alien-Invasion
python experiment

项目简介
这是一个基于 Pygame 的《外星人入侵》游戏扩展版本，在原版游戏基础上增加了持久化最高分记录和外星人射击功能，使游戏更具挑战性和可玩性。

功能特性

基础功能

1、玩家控制飞船左右移动并发射子弹

2、消灭外星人获得分数

3、多关卡系统，随关卡提升游戏难度

4、生命值系统，被撞击或击中会损失生命

扩展功能

1、持久化最高分 - 游戏最高分会自动保存到文件，重启游戏不会丢失

2、外星人射击 - 外星人会随机向下发射红色子弹，增加游戏挑战性

3、智能难度 - 外星人射击频率和子弹速度随关卡提升

系统要求

Python 3.6 或更高版本

Pygame 2.0+ 库
